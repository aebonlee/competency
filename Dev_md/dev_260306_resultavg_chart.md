## 1. 개요

`/results/average` (ResultAvg.jsx) 페이지를 레거시 JSP(`resultAvg.jsp`)와 동일한 구조로 재작성하였다.
기존 테이블 기반 표시에서 Chart.js 수평 바 차트 기반으로 전환.

---

## 2. 변경 내용

### 2.1 ResultAvg.jsx — 차트 기반 통계 페이지 재작성

**레거시 JSP 구조 재현:**
- **인구통계학적 통계** (나이별): 로그인 사용자의 나이 그룹 평균 수평 바 차트
- **성별 통계**: 로그인 사용자의 성별 그룹 평균 수평 바 차트
- **24 직무별 통계**: 24개 직무 카테고리 버튼 → 클릭 시 해당 직무 평균 차트 표시

**기술 구현:**
- `react-chartjs-2` `Bar` 컴포넌트 + `indexAxis: 'y'` (수평 바)
- 8대 역량별 고유 색상 유지 (COMPETENCY_COLORS)
- `useAuth()`로 현재 사용자의 성별/나이 그룹 참조
- 데이터 없는 그룹은 레거시와 동일하게 랜덤 데이터(160~400) 생성

**직무 24 카테고리 (POSITION_LIST 기반):**
- 레거시 NCS 24분류에서 현행 24개 직무 코드로 전환
- 토글 버튼 클릭 → 해당 직무 차트 표시 + 스크롤
